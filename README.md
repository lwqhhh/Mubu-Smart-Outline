# Mubu-Smart-Outline
Solved the issue of excessive content in Mind Map outline mode being difficult to index.  
**How to Use**
1. Install [**Tampermonkey**](https://microsoftedge.microsoft.com/addons/detail/%E7%AF%A1%E6%94%B9%E7%8C%B4/iikmkjmpaadaobahmlepeloendndfphd)
2.  Enter **OutlineScript.user.js**
3. Click **RAW** to install
